## Необходимые библиотеки для запуска

```
sudo apt install libiio-dev libncurses5-dev libboost-program-options-dev libgpm-dev

sudo apt install libqt5charts5-dev

```

## Сборка проекта

```
cd QT_SDR
```
```
cmake build/
```
```
make
```
```
./qt
```
